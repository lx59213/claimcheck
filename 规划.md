# ClaimCheck 产品规划

> **一句话**：电商详情页风险管理系统——售前查合规，售后解客诉

---

## 一、为什么做这个产品

### 真实痛点

**2025 年 11 月，影视飓风“全压胶”风波**：

- 详情页写“全压胶”，消费者理解为“每一处都压胶”

- 实际是行业术语，指“主要接缝处全压胶”（口袋不压胶是行业共识）

- 品牌没造假，但被误读，引发舆论风波，被迫两次公开自证

**类似案例每天都在发生**：

- “无氟涂层”“90 白鸭绒”“防水指数 10000mm”

- 行业术语 vs 消费者理解 = **认知鸿沟** = 舆论风险

### 现有方案的问题

**售前审核**：

- 人工审核慢（2 小时+），容易漏

- 只能查违禁词，查不出“认知鸿沟”

**售后客服**：

- 客户投诉时，客服现场编话术，容易说错

- 不知道详情页哪些词有风险，解释不清楚

### 我们的解决方案

**一个系统，两个场景，数据打通**：

1. **售前查合规**：AI 扫描详情页，发现违禁词 + 认知鸿沟

2. **售后解客诉**：客户投诉时，AI 调用售前分析，生成专业回复话术

**核心价值**：

- 售前预防风险（上架前就改好）

- 售后快速应对（10 秒生成话术）

- 数据复用（不需要客服再问一遍 AI）

## 二、核心原则（必须遵守）

### 1. 简单直白粗暴

- 界面一眼看懂，不需要学习成本

- 文案直白：“售前查合规” vs “售后解客诉”

- 流程简单：3 步完成任务

### 2. 花小力办大事

- Demo 体量小，开发成本低

- 不做复杂功能（如实时对话、复杂权限管理）

### 3. 洞察准

- 核心价值是“认知鸿沟检测”，不只是查违禁词

- AI 要能模拟消费者视角，判断术语是否容易误读

### 4. 易于验证

- AI 输出结果要直观、明确

- 高亮标注：红色 = 违禁词，黄色 = 认知鸿沟

- 每个风险点都给出：原因 + 修改建议

### 5. 好演示

- 用影视飓风真实案例（STORMCREW 冲锋衣）

- 视频 90 秒讲清楚：痛点 → 售前 → 售后 → 价值

- 界面要简洁，录屏时一眼能看懂

### 6. 越用越懂 + AI 主动预警（数据沉淀）

**为什么重要**：

- 风险不应该只靠人主动去查，AI 要有主动性

- 产品团队容易被“行业共识”蒙蔽，忽略消费者视角

- 需要系统主动联想、主动提醒

**系统会做什么**：

- 记录每次审核发现的风险、每次客诉的问题

- 公司信息沉淀：哪些词容易被误读、客诉高频问题是什么

- **主动联想**：

  - 公司历史：这个词/这类表达，你们之前出过问题

  - 行业案例：同类商品在网上有类似舆论风波（如其他品牌的“全压胶”争议）

  - 认知鸿沟：这个行业共识，消费者可能不懂

**最终效果**：

- 上新品时，AI 主动提示：“这个词建议改，你们去年因为类似表达被投诉过 3 次”

- 不需要人记得所有历史问题，AI 会主动关联

- 越用越懂这家公司的风险模式，预警越来越准

**这是未来的大趋势**：从“被动工具”到“主动助手”

## 三、要达到的效果

### 售前场景

1. 运营粘贴详情页文案

2. AI 扫描，30 秒输出结果

3. 看到高亮标注（红色违禁词 + 黄色认知鸿沟）+ 修改建议

4. 一键生成修改后文案

### 售后场景

1. 客服输入商品名 + 客户问题

2. AI 调取售前审核数据，10 秒生成回复话术

3. 话术结构：共情 → 解释行业标准 → 提供证明 → 给出方案

4. 一键复制，发给客户

### 数据沉淀场景

1. 系统记录所有审核风险 + 客诉问题

2. 下次上新品，AI 主动提示：

   - “这个词你们之前出过问题”

   - “同类商品在网上有类似舆论风波”

   - “这个行业术语消费者可能误读”

3. 越用越懂公司的风险模式

## 四、演示方案

### Demo 数据

- **售前输入**：STORMCREW 冲锋衣真实详情页文案（从天猫复制）

- **售前输出**：AI 扫描结果（3 个高危违禁词 + 5 个中危认知鸿沟）

- **售后输入**：客户投诉“你们详情页写的全压胶，为什么口袋没压胶？”

- **售后输出**：AI 生成的专业回复话术（共情 + 解释 + 证明 + 方案）

### 视频脚本（90 秒）

1. **痛点**（0-20s）：影视飓风“全压胶”风波 + 李佳琦事件

2. **售前**（20-45s）：运营扫描详情页，AI 发现风险，一键修改

3. **售后**（45-70s）：客户投诉，客服打开系统，10 秒生成话术

4. **价值**（70-90s）：对比传统方式 + 强调数据沉淀和 AI 主动性

## 五、成功关键

1. **认知鸿沟检测要准**：这是核心差异化价值，不只是查违禁词

2. **AI 主动性要强**：不是人问了才答，而是主动联想、主动提醒

3. **数据沉淀要有用**：越用越懂公司，预警越来越准

4. **界面要简洁**：双入口一眼看懂，流程清晰

5. **演示要真实**：用影视飓风真实案例，说服力最强
