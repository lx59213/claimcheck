# ClaimCheck · 电商详情页风险管理系统

AI 驱动的电商文案风险审查工具原型，用于演示如何在售前防范宣传风险、售后智能应对客诉。

## 快速启动

```bash
cd claimcheck
npx http-server -p 8765 -c-1
```

浏览器访问 `http://127.0.0.1:8765/`

## 功能概览

| 页面 | 功能 |
|------|------|
| 概览 `index.html` | 商品风险仪表盘、AI 动态（监控+洞察+学习合一） |
| 售前扫描 `presale.html` | 上传文案/图片 → 三维度扫描（合规/打假/认知） → 分级标注 → 修复建议 |
| 售后话术 `aftersale.html` | 粘贴客户投诉 → 关联售前风险 → 生成应对话术 |
| 知识库 `kb.html` | 商品级知识库（法规/标准/案例/内部/专家），类型筛选，关联风险项，企业知识源占位 |
| 风险总览 `archive.html` | SKU概况卡片（可折叠+知识依据计数）、修复追踪（高/中/低三列可折叠+筛选） |
| 风险历史 `history.html` | 单商品趋势图、历史扫描记录、风险采用状态 |

## 演示商品

|-----|------|-----------|
| 冲锋衣 | STORMCREW 三合一冲锋衣 | "全压胶"宣称不准确、无氟涂层技术悖论 |

## 技术栈

- 纯 HTML/CSS/JS，零构建依赖
- `localStorage` 持久化状态（`cc_` 前缀）
- Phosphor Icons + Plus Jakarta Sans + Noto Sans SC
- Claude 暖色调设计语言

## 隐藏功能

- **重置数据**：双击侧边栏底部 "ClaimCheck v0.1 Demo"
- **切换商品**：点击侧边栏当前商品名称
- **搜索**：侧边栏搜索入口，支持按类型筛选（含知识库）
- **知识库**：每个商品的风险判定依据，从售前风险卡可直接跳转定位
